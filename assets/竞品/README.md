# 竞品截图池

这个目录用来放竞品与截图池来源的页面层截图资产，包括 `美叶` 收集来的图。

它的职责不是替代 `learning/` 里的机制判断，而是给 `woodstuck` 提供可复用的界面层参考。

## 使用原则

- 图放这里，判断写到 `learning/SESSION_LOG.md`、`learning/TREND_LOG.md` 或 `learning/PROFILE.md`
- 先按玩法结构归类，再看视觉表现，不要只按“高级感 / 科技感 / 渐变风格”分组
- 截图只能证明 `前台长什么样`，不能单独证明 `是不是最新机制 / 上线时间 / 官方主推强度`
- 如果截图只是临时看一眼、不值得长期复用，可以不落库

## 目录结构

- `00-待归类/`

- `05-活动运营/`
- `06-规模扩张/`

## 归类口径

### `00-待归类`

先放还没看清玩法归属的截图，后续再挪走。

### `01-发现与分发`

适合放首页推荐、搜索发现、频道入口、榜单、歌单分发、内容组织方式。

### `02-播放与沉浸`

适合放播放器、歌词页、电台页、听书播放页、沉浸态细节、自定义播放体验。

### `03-免费模式`

适合放免费听入口、轻任务、拉新承接、可探索但受控的体验设计。

### `04-会员转化`

适合放会员中心、开通会员、权益表达、特权详情、升级承接、付费理由展示。

### `05-活动运营`

适合放活动弹窗、预约页、节日运营、应援玩法、抽奖任务、短期运营专题。

### `06-规模扩张`

适合放分享裂变、邀请机制、青少年/家庭模式、校园场景、创作者扩张或生态入口。

## 文件命名

推荐格式：

`平台-页面分类-玩法关键词-日期.png`

例如：

- `QQ音乐-播放器-3D播放器-2026-06-17.png`
- `汽水音乐-活动运营-音乐CPTI测试-2026-06-17.png`
- `网易云-会员转化-黑胶权益页-2026-06-17.png`

## 调用方法

当任务提到 `美叶`、`截图池`、`页面灵感`、`播放器截图`、`会员中心截图`、`活动弹窗截图` 时，先看这里有没有已归档分组，再决定是否补抓新图。
