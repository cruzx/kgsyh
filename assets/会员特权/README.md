# 会员特权资料包

这个目录是 `会员特权 / 概念版会员 / 付费链路` 相关任务的资料包入口。

## 当前约定

- 先看这个 README，再决定要不要继续翻源文件
- 如果目录里已经有 PDF、表格、截图，就按这里的索引调用
- 如果源文件暂时不在仓库里，不要假装它们存在，直接回退到稳定规则和研究文件

## 资料包结构

常见源文件会包括：

- `特权功能梳理.pdf`
- `概念版-会员特权表.xlsx`

它们适合分别回答：

- `转化主力 / 圈人 / 触点 / 试用到期拦截 / 升级逻辑`
- `特权矩阵 / 页面归属 / 数据排序 / 权益差异`

## 当前缺失时的回退口径
