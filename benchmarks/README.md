# benchmarks

这个目录放 `大盘数据 / 复盘结论 / 实验结果 / 点击基线 / 模块表现证据`。

和 `research/` 的区别：

- `research/`
  - 更偏用户洞察、行业趋势、方法论、行为规律
- `benchmarks/`
  - 更偏业务数据、实验结果、模块收益、基线和复盘证据

适合放进 `benchmarks/` 的内容：

- 版本改版复盘
- A/B 实验结论
- 首页 / 播放页 / 会员 / 概念版模块表现数据
- 渠道点击基线
- 灰度结果与份额变化说明
