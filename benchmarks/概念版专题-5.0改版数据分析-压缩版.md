# 概念版专题：5.0 改版数据分析 - 压缩版

来源：

- `【概念版】5.0改版数据分析1.0（20251028）.docx`

适用场景：

- 概念版首页改版复盘
- 首页卡片策略判断
- 我页份额变化解释
- 播放页改版收益判断
- 改版后数据表达、汇报、拍板

## 一句话结论

概念版 5.0 改版的整体信号偏正向：

- 大部分核心指标为 `微正向增长`
- 首页个推卡片和红心 radio 带来更明显的正向变化
- 我页份额下降属于首页能力增强后的结构性转移，`符合预期`
- 新播放页的入口点击表现更强，尤其 `评论入口`

## 总体实验结论

### 核心指标

数据源：

- `9.5 - 9.14` 灰度期间
- 对照组 = `4.0 首页`
- 实验组 = `5.0 首页`
- 我页均为 `5.0`

结论：

- 大部分指标均为 `微正向增长`
- `会员收入` 微负向
  - 由于大盘趋势无影响
  - 且本次无相关改动模块
  - 文档判断更接近 `正常波动`
  - 但注明：`未空跑，待准确归因`

### 播放与收藏

数据源：

- `9.6 - 9.15`
- 对照组 = `4.0 首页`
- 实验组 = `5.0 首页`
- 我页均为 `5.0`

结论：

- 实验组对比对照组：
  - 播放率微正向
  - 收藏率微正向
- 同样标注：`未空跑，待准确归因`

表格数据：

- 对照组播放率：`90.60%`
- 实验组播放率：`90.74%`
- 差值：`+0.14%`
- 对照组收藏率：`21.43%`
- 实验组收藏率：`22.59%`
- 差值：`+1.16%`

## 首页结论

### 个推卡片整体表现

数据源：

- `9.5 - 9.15` 灰度数据
- 对照组 = `4.0 首页`
- 实验组 = `5.0 首页`
- 我页均为 `5.0`

结论：

- 个推卡片播放份额和收藏份额均有提升
- 个推卡片播放量与收藏量也均有提升

表格数据：

- 播放份额：
  - 对照组 `6.28%`
  - 实验组 `6.96%`
  - 差值 `+0.68%`
- 收藏份额：
  - 对照组 `10.84%`
  - 实验组 `13.22%`
  - 差值 `+2.38%`
- 播放 pv：
  - 对照组 `16.63w`
  - 实验组 `18.70w`
  - 差值 `+2.07w`
- 收藏 pv：
  - 对照组 `1.04w`
  - 实验组 `1.47w`
  - 差值 `+0.43w`

判断：

- 首页改版后，个推卡片是这轮改版最明确的正向模块之一

### 各卡片细分表现

文档强调：

- 不同卡片效果差异明显
- `需要具体卡片具体分析`
- 标黄项为负增长

#### 负向项

- `VIP专属推荐`
  - 点击率 `-3%`
  - 播放率 `-3.6%`
  - 优化方向：
    - 增加 `播放按钮图标`
    - 增加歌曲数量露出
    - 保证露出 `3 首`

- `喜欢「标签」的 Ta 也喜欢`
  - 点击率 `-2.2%`
  - 播放率 `-2.5%`
  - 优化方向：
    - 增加 `播放按钮图标`
    - 优化卡片样式
    - 如每首歌都露出图

- `小众宝藏佳作`
  - 点击率 `-0.3%`
  - 播放率 `-0.9%`
  - 判断：
    - 可持续观察
  - 优化方向：
    - 增加 `播放按钮图标`
    - 优化卡片样式
    - 保证露出 `3 首`

#### 正向但有局部波动项

- `曲风盲盒`
  - 人均播放时长下降 `3.5min`
  - 点击率和播放率提升
  - 判断：
    - 持续观察

- `私人专属好歌 & 概念er新推`
  - 人均播放时长下降 `2min`
  - 点击率和播放率提升
  - 判断：
    - 持续观察

## 我页结论

### 不区分版本看覆盖率

时间：

- `10.10 - 10.20`
- 5.0 版本覆盖率 `67%`

对比：

- `8.20 - 8.30` 仅 4.0 版本

结论：

- 播放份额和收藏份额 `无明显差异`

表格数据：

- 播放份额：
  - `65.1% -> 65.2%`
  - 差值 `+0.1%`
- 收藏份额：
  - `38.6% -> 38.4%`
  - 差值 `-0.2%`

### 区分版本看 5.0

对比：

- `10.10 - 10.20` 限定仅 5.0 版本
- 对比 `8 月整月` 数据

结论：

- 播放份额 `-1.4%`
- 收藏份额 `-1.7%`

文档解释：

- 首页优化后
  - 个推卡片份额提升
  - 红心 radio 份额提升
- 因此我页份额下降，`符合预期`

判断：

- 这是典型的结构性份额转移
- 不应直接被解释成我页体验失败

## 播放页结论

数据源：

- `9.20 - 10.9`
- 三个播放器点击率数据
- 不区分版本

结论：

- 新播放页在页面、UI 按钮、交互调整后
  - 液体胶片各入口按钮点击率更高
  - `评论入口` 点击率比其他两个播放器高 `1 倍以上`

优化方向：

- 旧播放页的 UI 按钮交互，陆续优化为新版

辅助数据：

- 评论页曝光量随 5.0 覆盖率提升而提升

表格数据：

- 10.1 - 10.20 日均：
  - uv `789,443`
  - pv `3,901,399`
- 9 月日均：
  - uv `702,263`
  - pv `3,238,437`
- 9.5 - 9.20 日均：
  - uv `693,516`
  - pv `3,210,967`

判断：

- 新播放页不仅入口点击更强
- 还带动了评论页曝光提升

## 明确的优化动作

### 首页卡片层

文档里最明确的动作是：

- `需要保留收藏按钮`

同时对于若干负向卡片，统一出现的优化思路包括：

- 增加 `播放按钮图标`
- 增加歌曲数量露出
- 保证至少露出 `3 首`
- 卡片内容露出更直接，例如每首歌露图

## 最值得记住的 6 条判断

1. 5.0 改版整体偏正向，不是大幅收益型改版，而是 `微正向优化`
2. 首页个推卡片和红心 radio 是最明确的收益模块
3. 我页份额下降应理解为首页能力增强后的结构性转移，而不是单独失败
4. 新播放页的入口交互更强，尤其 `评论` 入口优势明显
5. 卡片改版里最共性的优化不是换概念，而是 `补直接动作信号和内容露出`
6. 这类改版汇报时，要特别强调 `未空跑，待准确归因`，避免把相关性直接写成因果

## 使用提醒

- 这份文档更适合作为：
  - 概念版 5.0 改版复盘依据
  - 首页卡片与播放页优化判断依据
  - 汇报时的结构性解释材料
